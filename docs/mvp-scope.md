# MVP Scope

This document will track the MVP scope for PitWall.

For the current full planning reference, see:

- [`project-plan.md`](./project-plan.md)

## Must Have

- Connection / session status
- Live timing tower
- Live track map / driver position overlay
- Driver focus panel
- Race-control feed

## Should Have

- Tyre and stint information
- Weather panel
- Basic selected-driver telemetry

## Not MVP

- Historical replay
- User accounts
- Paid subscriptions
- Strategy predictions
- Advanced overlays
